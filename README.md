# QSPI Interface for Cocotb

> [!WARNING]
> This is work in progress - may not work as intended.

Fork of the SPI-package to add support for QSPI simulation.  
For some more information check out the [original project](https://github.com/schang412/cocotbext-spi).  

## Installation

Installation from pip:

```bash
pip install git+https://github.com/timo-braungardt/cocotbext-qspi.git
```
